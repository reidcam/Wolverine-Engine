void LuaAPI::DeleteLuaTable ( std::shared_ptr< sol::table >  table)
static

Deletes a table on the lua stack

Note: does not actualy delete a table from the lua stack as only lua's garbage collection can do that. Instead, this function clears all data from the table and locks it in read-only. This is to make any lingering references to it in other scripts useless, and prevent zombie components from causing undefined behavior. DO NOT EXPOSE TO LUA, THIS IS AN ENGINE TOOL ONLY!
